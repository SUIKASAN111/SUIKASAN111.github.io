Best Visual Studio Setup for C++ Projects - The Cherno

2022-12-03 C/CPP setup 106 words
C/C++ tips setup

这篇文章是对The Cherno对Visual Studio文件组织形式自定义设置的分享。因为设置只对当前项目生效，所以每次在新项目上工作必须重新设置，为防止自己忘记，特别将其记录在此。

右击Project，选择Properties打开配置设置
在Configuration Properties->General中将
Output Directory路径改为$(SolutionDir)bin\$(Platform)\$(Configuration)\
Intermediate Directory路径改为$(SolutionDir)bin\intermediates\$(Platform)\$(Configuration)\
